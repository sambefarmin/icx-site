Methodology
How ICX Calculates Each Rate
Transparent. Repeatable. Independent of hardware cost models. Each index uses the same methodology applied to its own provider panel.
01

Collect

Automated scrapers run hourly, fetching executable on-demand pricing from a diversified panel of US-based cloud providers. Multi-GPU clusters normalized to per-GPU equivalent.

02

Normalize

Each index anchors to a specific Standard Compute Unit — e.g. 1× NVIDIA H100 SXM5 80GB (NVLink), US-located, on-demand, hourly provisionable. PCIe and SXM5 variants are tracked as separate indexes. All prices in $/GPU-hour.

03

Clean

Remove unavailable providers. Require a minimum panel (8 providers for H100 SXM5, 3 for H100 PCIe) — rate not published if panel is smaller. Trim the bottom and top 15% of remaining prices to remove statistical outliers.

04

Publish

Rates updated hourly from live scrapes. Each computation uses only currently-available providers, so the rate reflects executable spot prices — not stale listings. Each GPU type is stored as an independent time series.

05

Price ≠ Cost

ICX reflects transaction prices, not hardware ASP or production expense. Commodity benchmarks measure markets, not drilling costs. The rate anchors procurement — it is not a cost model.

06

Governance

Published methodology. Documented update process. Minimum data requirements enforced in code. Variant separation documented and applied consistently. The remedy for data risk is governance, not complexity.

Strategic Sequence
Benchmark First. Then Everything Else.
Derivatives cannot precede credibility. ICX follows the established path: measure → adopt → structure → financialize.
Phase 1 — Now

Measurement

Publish consistent hourly reference rates across GPU variants. Establish volatility patterns, seasonal shifts, and supply tightness in neutral instruments.

Phase 2

Adoption

Enterprises benchmark vendor quotes to ICX. Procurement teams embed ICX in forecasts. The signal: "ICX plus spread" enters the market vocabulary.

Phase 3

Structure

Term expectations, forward curves, and realized volatility measures derived from observed spot data. Analytic layers, not separate businesses.

Phase 4

Financialization

Bilateral swaps. Cash-settled futures. Only after the benchmark is sufficiently trusted and liquid. Trust is the prerequisite, not the product.
